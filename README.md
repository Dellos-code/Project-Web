# UniBite 🍔🎓
**Student Food Sharing Platform**

Το UniBite είναι μια πλατφόρμα διαμοιρασμού φαγητού για την πανεπιστημιακή κοινότητα. Στόχος του είναι να συνδέσει φοιτητές που μαγειρεύουν και έχουν περισσευάμενες μερίδες, με συμφοιτητές τους που αναζητούν ένα σπιτικό γεύμα μέσω ενός συστήματος ανταλλαγής πόντων (credits).

## 🛠️ Τεχνολογίες (Stack)
* **Backend:** Node.js, Express.js
* **Database:** SQLite (Zero-setup, ACID Transactions)
* **Frontend:** Vanilla JavaScript, HTML, CSS, Leaflet.js (Map)
